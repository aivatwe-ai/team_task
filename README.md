# team_task
### Обязательная часть
За это время вы подготовите собственный аналитический отчёт. Вы проанализируете данные из базы MovieLens. В итоге у вас будет два файла: movielens_analysis.py и movielens_report.ipynb. В первом файле вы создадите собственный модуль с классами и методами. Второй файл содержит сам отчёт, созданный с использованием только вашего модуля.

Модуль
Помните, что цель спешки — укрепить ваши навыки.
Постарайтесь применить как можно больше из того, чему вы научились за предыдущие дни.

Используйте уменьшенную версию набора данных MovieLens, скачайте его, пожалуйста. Используйте первые 1000 значений из набора данных.
 Внимательно прочитайте README.txt файл. Сосредоточьтесь на структуре файлов. 
В вашем модуле вам нужно будет создать четыре класса, соответствующих четырём файлам с данными, а также один класс для тестирования.
Приведённые ниже классы и методы являются обязательными, но вы можете добавить всё, что вам нужно.

Классы Ratings, Tags, Movies и Links можно найти в примерах кода.
Классовые тесты:
Создайте тесты с использованием PyTest для каждого метода вышеперечисленных классов.
Они должны проверять:

если методы возвращают данные правильного типа,
если элементы списка имеют правильный тип данных,
если возвращённые данные отсортированы правильно.

Запустите тесты, прежде чем переходить к следующему этапу.

Отчет
 Подготовьте отчет, используя только классы и методы из movielens_analysis.py.
Сделайте это в Jupyter Notebook. Это отличный инструмент, особенно для специалистов по обработке данных. Он позволяет работать с кодом в интерактивном режиме, запуская и перезапуская разные ячейки с разными значениями. Вам не придётся заново запускать весь код с самого начала. Вы также можете добавлять текст в ячейки, что очень удобно для создания отчётов. Установите его в своей среде.
В этом разделе мы предоставим вам больше свободы. Мы не будем определять структуру вашего отчёта. Цель отчёта — рассказать интересную историю о наборе данных MovieLens. Найдите подходящую структуру и последовательность.
Единственными ограничениями являются:

Вы должны использовать все методы в movielens_analysis.py, кроме класса Tests.
 Каждая ячейка в вашем блокноте должна содержать магическую команду %timeit. 
Все остальные импорты и встроенные функции запрещены. Если они вам нужны, добавьте их в свой модуль заранее.
